본 페이지는 지수 함수 게임 사용자가 본인의 계정과 관련 데이터의 삭제를 요청하는 방법을 안내합니다. 관련 법령(개인정보 보호법) 및 Google Play 사용자 데이터 정책에 따라 모든 요청은 지체 없이 처리됩니다.

1. 계정 삭제 요청 방법

아래 절차에 따라 이메일로 요청해 주시면 됩니다.

  1. 아래 이메일 주소로 메일을 보냅니다.
  2. 메일 제목에 [지수 함수 게임] 계정 삭제 요청이라고 기재합니다.
  3. 메일 본문에 다음 정보를 포함합니다.
    • 로그인에 사용한 Google 계정 이메일 주소 (Google 로그인 사용자)
      또는 앱 내 표시되는 게스트 ID (게스트 사용자, 설정 화면에서 확인 가능)
    • 앱 내 닉네임(설정한 경우)
  4. 요청 접수 후 영업일 기준 7일 이내에 처리하고 확인 메일을 회신합니다.

4. 게스트 모드 사용자

게스트 모드로 이용한 데이터는 별도 요청 없이도 마지막 접속일로부터 90일 후 자동 삭제됩니다. 즉시 삭제를 원하시면 위 1번 절차에 따라 게스트 ID를 함께 보내주세요.

5. 자주 묻는 질문

Q. 앱에서 직접 계정을 삭제할 수는 없나요?

현재 버전에서는 앱 내 일괄 삭제 기능을 제공하지 않으며, 이메일 요청 방식으로 처리하고 있습니다. 향후 업데이트에서 앱 내 삭제 기능 추가를 검토 중입니다.

Q. 계정을 삭제하지 않고 게임 기록만 초기화할 수 있나요?

현재는 부분 삭제 기능을 제공하지 않습니다. 계정 전체 삭제 후 다시 로그인하시면 새로 시작할 수 있습니다.

Q. 삭제 후 복구가 가능한가요?

삭제된 데이터는 복구할 수 없습니다. 신중히 요청해 주세요.
