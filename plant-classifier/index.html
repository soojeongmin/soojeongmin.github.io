식물카드

한국 자생식물을 사진과 분류 체계로 익히는 플래시카드 · 퀴즈 학습 앱

소개

국립수목원 국가표준식물목록을 바탕으로 한국 자생식물을 사진과 분류 체계(과·속·국명·학명)로 학습하는 iOS·Android 앱입니다. 플래시카드로 외우고, 퀴즈로 점검하고, 간격 반복 복습으로 오늘 외울 카드를 자동으로 골라줍니다.

특징

  • 플래시카드 — 카드를 넘기며 알아요/모름으로 익히기
  • 퀴즈 — 사진을 보고 이름·분류를 맞추는 객관식·주관식 문제
  • 간격 반복 복습(SRS) — 정답 여부에 따라 다음 복습 시점을 자동 계산
  • 혼동 드릴 — 헷갈리는 비슷한 종을 나란히 비교
  • 분류 탐색 — 과·속으로 묶어 계통 둘러보기
  • 학습 세트 — 친숙한 식물(입문)·특산식물·전체·안 본·모름·오답 카드
  • 과별 학습 · 테마별 세트(대형/소형/털/가시/덩굴/산·고산/물·습지 등)
  • 연속 스트릭 · 일일 학습 알림 · 명예의 전당(랭킹)
  • 다크 모드 / 한국어·영어 지원
  • 로그인 없이도 무료 · Apple·Google 로그인 시 기기 간 동기화

데이터 출처

  • 식물 분류 체계 · 국명 · 학명: 산림청 국립수목원 국가표준식물목록 (data.go.kr)
  • 식물 사진: 산림청 국립수목원 표준식물목록 이미지정보 (공공누리 제4유형)
