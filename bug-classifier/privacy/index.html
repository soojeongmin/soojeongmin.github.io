1. 수집하는 개인정보 항목

1-1. 소셜 로그인(Apple / Google) 이용 시

항목수집 시점이용 목적

사용자 ID (Apple / Google)최초 로그인 시계정 인증

iOS에서 "Apple로 로그인" 사용 시 Apple의 이메일 가리기(비공개 이메일 릴레이)를 선택하면 실제 이메일 대신 익명 릴레이 주소만 전달됩니다. 본 앱은 Apple·Google이 제공하는 이름은 별도로 저장하지 않습니다.

항목수집 시점이용 목적

익명 이용 시에는 이메일·이름 등 개인을 직접 식별할 수 있는 정보를 수집하지 않습니다.

1-3. 모든 이용자 공통 수집 정보

항목이용 목적

본 앱은 별도의 광고 SDK 또는 분석 SDK를 사용하지 않으며, IP 주소·기기 광고 식별자(GAID)·위치정보·연락처·사진을 수집하지 않습니다.

2. 개인정보의 보유 및 이용 기간

  • 소셜 로그인(Apple/Google) 사용자: 회원 탈퇴 또는 삭제 요청 시까지 보관
  • 익명 이용 사용자: 앱 삭제 시 또는 마지막 접속일로부터 180일 후 자동 정리

이용자는 언제든지 본 앱 설정 화면의 "계정 탈퇴" 또는 아래 연락처를 통해 본인의 정보 삭제를 요청할 수 있으며, 요청 시 지체 없이 모든 데이터를 파기합니다.
